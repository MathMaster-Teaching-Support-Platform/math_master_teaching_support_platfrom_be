- student-chatbot.puml
- payment-deposit-webhook-wallet.puml

## Import to draw.io

1. Open draw.io canvas.
2. Arrange -> Insert -> Advanced -> PlantUML.
3. Paste content from one `.puml` file.
4. Click Insert.
